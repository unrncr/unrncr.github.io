Sources

Having useful sources is important. Below are a few of the sources that help the team to understand both the project topic and its implementation.

The Problem Domain

The problem domain for this project is computational neuroscience. The project deals with both the graphical representation of data and the structure of brain models. Therefore, a book in the problem domain must cover many aspects of computational neuroscience.

The book The Fundamentals of Computational Neuroscience: Ed 2 will be a helpful resource in this field of study.

Description: The book is an introductory material for the field of computational neuroscience. It describes the different functions of the brain in order to introduce the reader to general neuroscience knowledge. More importantly, the book explains the types of models used when creating computational versions of a brain, tying in the computational aspect of the field.

Journal Articles

  • Hoang, R.V., Tanna, D., Jayet Bray, L.C., Dascalu, S.M., and F. C. Harris, Jr. (2013). A Novel CPU/GPU Simulation Environment for Large-Scale Biologically Realistic Neural Modeling. Frontiers in Neuroinformatics, vol. 7, no. 19.
     Description: The article details the issue of computation time for brain simulations and describes the Neo Cortical Simulator NCS from the University of Nevada Brain Computation Lab. It describes the approach to designing the simulator, the built in neuron model types, and its use of the graphical processing unit to more efficiently run simulations.

    Description: The article is useful because it evaluates the many different parts of brain simulations such as strategies, algorithms, and neuron types. The evaluations include comparisons between the uses of different model types with different simulation types. It includes ideas on how to compare the simulation report data and make conclusions using the data.
